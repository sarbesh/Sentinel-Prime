# Unified Web/Mobile UI (React Native)

## Purpose
Centralize all configuration, reporting, alerts, and device management through a single, unified codebase—exporting to web and native apps.

## Getting Started
- Scaffold with React Native, use Expo for easy web export
- See ADR.md for rationale

## API/Integration
- Connects to backend API for all modules (scanners, honeypot, IPS/IDS, device monitor)
- Authentication/login, role management, dashboard, device/block/reports

## Contributing
- Add screens, widgets, notifications, improve UX, integrate new features as modules are added.
