# Backend/API

## Purpose
Central API for all modules: scanners, device monitor, honeypot, IPS/IDS, reporting, and unified authentication/storage.

## Getting Started
- Scaffold in preferred language (Python FastAPI suggested)
- See ADR.md for decision rationale

## API/Integration
- REST or GraphQL API endpoints for all core functionalities
- Manages DB storage, modular triggers, and notification workflows

## Contributing
- Extend API for new modules, add endpoints, improve DB schema
