# Honeypot Module (OpenCanary/Dionaea)

## Purpose
Emulate vulnerable devices/services, log attack attempts, integrate with threat intelligence and UI.

## Getting Started
- See ADR.md for rationale and design.
- Launch with Docker Compose, or directly as a container.

## API/Integration
- Logs to backend/API, triggers block/unblock actions

## How to contribute
- Update docs, add new protocol emulations, improve attack reporting, or integrate new honeypot tools.
