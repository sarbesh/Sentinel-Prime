# Contributing to Sentinel Prime

Thank you for considering contributing to Sentinel Prime! We welcome contributions from the community.

## How to Contribute

1. Fork the repository
2. Create a new branch for your feature or bug fix
3. Make your changes
4. Ensure your code follows the existing style and conventions
5. Add tests if applicable
6. Submit a pull request

## Development Setup

For backend development:
- See `backend/README.md` for setup instructions

For web UI development:
- See `web-ui/README.md` for setup instructions

## Reporting Issues

Please use the GitHub issue tracker to report bugs or request features.
